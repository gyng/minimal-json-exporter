# minimal-json-exporter

Minimal JSON exporter for Prometheus. Flattens nested JSON.

## Requirements

- Python 3 with Flask

## Install

```bash
$ pip install -r requirements.txt
```

## Use

```
$ pip install flask
$ URL=http://x.x.x.x python app.py
```

# Docker

```
$ docker-compose up -d
```
